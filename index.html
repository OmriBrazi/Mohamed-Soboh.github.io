Introduction

The site we chose is - ASOS. Asos is a British e-commerce company, specializing in fashion and cosmetics.

The project's description

We wanted to focus on comparing prices between countries


Problems:

  • The search in Asos gives a price for the country we choose.
  • Price for each currency is different even if it is the same country.
  • There are no deliveries to Israel from all countries.

    • The solution:

  • Retrieving the list of countries and currencies of each country that exist on the site.
  • Searching for each size separately and combining the results.

    interesting technologies that you used in the project.

    We use Beautiful Soup It is a Python library that is used for web scraping purposes to pull the data out of HTML and XML files. It creates parse trees that are helpful to extract the data easily. Beautiful Soup provides a few simple methods and Pythonic idioms for navigating, searching, and modifying a parse tree, and it sits on top of popular Python parsers like lxml and html5lib, allowing users to try out different parsing strategies or trade speed for flexibility. Another Python library we used is currency_converter, it converts one currency to another and the default source is the European Central Bank.
